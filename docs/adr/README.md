# Architecture Decision Records

This directory records the heavier design decisions behind the pipeline, in
[ADR](https://adr.github.io/) format — one short file per decision, capturing the
**context**, the **decision**, and its **consequences** so the reasoning survives
after the diff is forgotten. Lighter decisions live in the README's
[design notes](../../README.md#design-notes).
